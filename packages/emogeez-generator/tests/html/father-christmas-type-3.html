Modifiers

The Santa Claus emoji supports skin tone modifiers. A yellow (or other non-human) skin tone should be shown by default, unless an emoji modifier is applied.

  • 🎅 Santa Claus
  • 🎅🏻 Santa Claus: Light Skin Tone
  • 🎅🏽 Santa Claus: Medium Skin Tone
  • 🎅🏾 Santa Claus: Medium-Dark Skin Tone
  • 🎅🏿 Santa Claus: Dark Skin Tone

Platforms without support for emoji modifiers display a missing symbol character (box with a cross, alien, or question mark in it) next to any modified emoji.
